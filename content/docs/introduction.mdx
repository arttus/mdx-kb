# Introduction

Welcome to the **MDX Knowledge Base** - a modern documentation platform built with cutting-edge web technologies.

## What is MDX Knowledge Base?

MDX Knowledge Base is a powerful documentation system that combines the simplicity of Markdown with the flexibility of React components through MDX. It's designed to help you create beautiful, interactive documentation with ease.

## Key Features

- **MDX Support**: Write in Markdown and use React components seamlessly
- **Syntax Highlighting**: Beautiful code blocks powered by Shiki
- **Mermaid Diagrams**: Create flowcharts, sequence diagrams, and more
- **Dark Mode**: Automatic theme switching with system preference detection
- **Fast & Modern**: Built with Next.js 15, React 19, and Tailwind CSS 4
- **Accessible**: WCAG compliant with keyboard navigation support

## Quick Example

Here's a simple code example with syntax highlighting:

```typescript title="example.ts"
interface User {
  id: number;
  name: string;
  email: string;
}

function greetUser(user: User): string {
  return `Hello, ${user.name}!`;
}

const user: User = {
  id: 1,
  name: "John Doe",
  email: "john@example.com"
};

console.log(greetUser(user));
```
