# Installation

Get started with MDX Knowledge Base in just a few minutes.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** 20.x or later
- **npm** or **yarn** or **pnpm**
- A code editor (we recommend VS Code)

## Clone the Repository

The fastest way to get started is to clone the repository:

<Tabs defaultValue="ssh">
  <TabsList>
    <TabsTrigger value="ssh">SSH</TabsTrigger>
    <TabsTrigger value="https">HTTPS</TabsTrigger>
    <TabsTrigger value="gh">GitHub CLI</TabsTrigger>
  </TabsList>
  <TabsContent value="ssh">
    ```bash
    git clone git@github.com:arttus/mdx-kb.git my-knowledge-base
    cd my-knowledge-base
    npm install
    ```
  </TabsContent>
  <TabsContent value="https">
    ```bash
    git clone https://github.com/arttus/mdx-kb.git my-knowledge-base
    cd my-knowledge-base
    npm install
    ```
  </TabsContent>
  <TabsContent value="gh">
    ```bash
    gh repo clone arttus/mdx-kb my-knowledge-base
    cd my-knowledge-base
    npm install
    ```
  </TabsContent>
</Tabs>

<Callout type="info">
  All dependencies are already configured. Just clone, install, and start writing!
</Callout>

## Start the Development Server

Run the development server to see your knowledge base in action:

<Tabs defaultValue="npm">
  <TabsList>
    <TabsTrigger value="npm">npm</TabsTrigger>
    <TabsTrigger value="yarn">Yarn</TabsTrigger>
    <TabsTrigger value="pnpm">pnpm</TabsTrigger>
  </TabsList>
  <TabsContent value="npm">
    ```bash
    npm run dev
    ```
  </TabsContent>
  <TabsContent value="yarn">
    ```bash
    yarn dev
    ```
  </TabsContent>
  <TabsContent value="pnpm">
    ```bash
    pnpm dev
    ```
  </TabsContent>
</Tabs>

Open [http://localhost:3000](http://localhost:3000) in your browser to see your knowledge base.
