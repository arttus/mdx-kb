# Installation

Get started with MDX Knowledge Base in just a few minutes.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** 20.x or later
- **npm** or **yarn** or **pnpm**
- A code editor (we recommend VS Code)

## Install Dependencies

Install the required packages for MDX, syntax highlighting, and UI components:

```bash
npm install @next/mdx @mdx-js/loader @mdx-js/react gray-matter
npm install remark-gfm remark-math rehype-slug rehype-autolink-headings rehype-pretty-code shiki
npm install mermaid next-themes lucide-react
npm install class-variance-authority clsx tailwind-merge
```

## Configure Next.js

Update your `next.config.ts` to support MDX:

```typescript title="next.config.ts"
import createMDX from '@next/mdx';

const nextConfig = {
  pageExtensions: ['js', 'jsx', 'md', 'mdx', 'ts', 'tsx'],
};

const withMDX = createMDX({
  extension: /\.mdx?$/,
  options: {
    remarkPlugins: [],
    rehypePlugins: [],
  },
});

export default withMDX(nextConfig);
```

## Project Structure

Your project should now have the following structure:

```
my-knowledge-base/
├── app/
│   ├── layout.tsx
│   ├── page.tsx
│   └── docs/
│       └── [...slug]/
│           └── page.tsx
├── components/
│   ├── ui/
│   └── mdx/
├── content/
│   └── docs/
│       └── *.mdx
├── lib/
└── package.json
```
